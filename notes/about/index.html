About

Camera-free live coding from Skreieweydo Labs — Common Lisp, functional patterns, ARM64 that touches the metal, and the math that explains the why. Every episode gets a notes page with code & timestamps.

Mission

Teach systems thinking through readable, test-first code. We ship tiny, correct demos; we prove ideas with small tests; and we explain performance with shapes, not folklore.

  • Clarity over cleverness — small seams, pure cores.
  • Reproducibility — deterministic builds, explicit tools.
  • Math where it helps — proofs, invariants, and quick heuristics.
What to expect
  • Live coding in Common Lisp with focused FP patterns.
  • Short tests first (FiveAM), property checks later.
  • Occasional ARM64 detours to “touch the metal.”
  • Notes posted within 24h: code, timestamps, and references.
